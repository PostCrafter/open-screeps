# is-creep-fatigued
> Is the creep fatigued

## Install
```sh
$ npm install @open-screeps/is-creep-fatigued
```

## Usage
```typescript
import { isCreepFatigued } from '@open-screeps/is-creep-fatigued';

if(!isCreepFatigued(creep)){
  creep.moveTo(target)
}
```
